# bisnisfotocopy
BisnisFotocopy adalah aplikasi desktop berbasis Windows yang dirancang untuk membantu pemilik usaha fotokopi dalam mengelola seluruh operasional bisnis secara terintegrasi. Sistem ini menggabungkan pencatatan transaksi, manajemen stok, keuangan, layanan, serta pelaporan dalam satu platform yang mudah digunakan.
Fitur Utama
📋 Dashboard bisnis interaktif
💰 Manajemen pemasukan dan pengeluaran
🖨️ Transaksi fotokopi, print, scan, laminating, jilid, dan layanan lainnya
📦 Manajemen stok ATK dan bahan habis pakai
🛒 Penjualan produk retail
👥 Data pelanggan dan supplier
📑 Riwayat transaksi lengkap
📊 Laporan keuangan otomatis
📈 Analisis laba rugi
💳 Pencatatan hutang dan piutang
🔧 Monitoring biaya operasional
🖥️ Multi-modul dalam satu aplikasi desktop
🔐 Sistem login dan hak akses pengguna
💾 Backup dan restore database
